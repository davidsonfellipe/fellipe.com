# www.fellipe.com

My personal website using jekyll-bootstrap

# steps to run

    gem install jekyll

    git clone https://github.com/davidsonfellipe/fellipe.com.git

    cd fellipe.com

    jekyll --server

On browser [localhost:4000](localhost:4000)


# dependencies

    gem install compass
