Finally, with the blog I'll have more space and flexibility to share past and future projects. I'm kicking things off with what was, until then, my most important work: my undergraduate thesis. The main goal was to design and implement a wireless communication system using Bluetooth technology and the PIC microcontroller, to support teaching activities and future Bluetooth-based projects at the University of Pernambuco.

## The objectives of the project were:

-   Design hardware that would serve as a module for developing applications that need Bluetooth.
-   Build a small API with basic functions to make it easier to work with Bluetooth data transmission using the PIC microcontroller.
