Principais pontos da Web Semântica?

  • web de documentos para web de dados
  • dados devem ser acessados usando a arquitetura da web, e.g. URI-s
  • dados deve ser relacionados uns aos outros não somente como documentos
  • documentos serem processados automaticamente de forma mais padronizada

RDF é uma aplicação XML?

Não. O modelo fundamento de RDF é independente de XML.

https://www.w3.org/RDF/

Mais conceitos sobre RDF

  • Suas recomendações foram publicadas em 2004, pelo W3C.
  • Não é uma linguagem, mas sim, um modelo.
  • Utilizada para descrever recursos na web
  • É um padrão para intercâmbio de dados na Web.
https://www.w3.org/RDF/

RDF têm três componentes básicos

  • Recurso: Qualquer coisa que pode conter uma URI, pode ser páginas da web, assim como outros elementos de um documento XML.
  • Propriedade: Um recurso que tenha um determinado nome e possa ser utilizado como uma propriedade.
  • Indicação: consiste na combinação de um recurso, de uma propriedade, e de um valor.
https://www.w3.org/RDF/

RDF Schema

  • é uma extensão semântica do RDF
  • classes hierarquicas e propriedades hierarquicas
  • tem várias abreviações RDFS, RDF(S), RDF-S, or RDF/S
  • A 1ª versão foi publicada, pelo W3C, em Abril/98
https://www.w3.org/TR/rdf-schema/

Schema

utilizados apenas para definir as propriedades dos recursos, mas também os tipos de recursos descritos.

https://www.w3.org/RDF/

Elemento raíz e namespace 

<?xml version="1.0"?>
<rdf:RDF xmlns:rdf="https://www.w3.org/1999/02/22-rdf-syntax-ns#"
         xmlns:rdfs="https://www.w3.org/2000/01/rdf-schema#"
         xml:base="http://www.animals.fake/animals#">
</rdf:RDF>
https://www.w3.org/TR/rdf-schema/

RDF description

rdf:about se refere a um recurso existente
rdf:ID criação de um recurso 

<?xml version="1.0"?>
<rdf:RDF xmlns:rdf="https://www.w3.org/1999/02/22-rdf-syntax-ns#"
         xmlns:rdfs="https://www.w3.org/2000/01/rdf-schema#"
         xml:base="http://www.animals.fake/animals#">
  <rdf:Description rdf:ID="animal">
    <rdf:type
       rdf:resource="https://www.w3.org/2000/01/rdf-schema#Class"/>
  </rdf:Description>
</rdf:RDF>
https://www.w3.org/TR/rdf-schema/

Exemplo simplificado de RDF Schema 

<?xml version="1.0"?>
<rdf:RDF xmlns:rdf="https://www.w3.org/1999/02/22-rdf-syntax-ns#"
         xmlns:rdfs="https://www.w3.org/2000/01/rdf-schema#"
         xml:base="http://www.animals.fake/animals#">
  <rdf:Description rdf:ID="animal">
    <rdf:type
     rdf:resource="https://www.w3.org/2000/01/rdf-schema#Class"/>
  </rdf:Description>
  <rdf:Description rdf:ID="horse">
    <rdf:type
     rdf:resource="https://www.w3.org/2000/01/rdf-schema#Class"/>
    <rdfs:subClassOf rdf:resource="#animal"/>
  </rdf:Description>
</rdf:RDF>
https://www.w3.org/TR/rdf-schema/

RDF Schema especificar convenções

  • Classe têm a primeira letra maiúscula
    Exemplo: rdfs:Resource
  • Propriedades não têm a primeira letra maiúscula
    Exemplo: rdfs:domain
  • Palavras adicionais ao nome são maiúsculas Exemplo: rdfs:subClassOf e rdfs:ConstraintProperty

RDF Classes - parte 1

rdfs:Resource Classe para recursos, ou seja qualquer coisa.
rdfs:Literal Classe para strings ou inteiros.
rdf:XMLLiteral Classe para valores literais de XML.
rdfs:Class Classes de classes.
rdf:Property Classe para propriedades RDF.
rdfs:Datatype Classe para tipos de dados RDF
rdf:Statement Classe para declarações de RDF.
https://www.w3.org/TR/rdf-schema/

RDF Classes - parte 2

rdf:Bag Classe para conteiners que não exigem ordenação.
rdf:Seq Classe para conteiners que exigem ordenação.
rdf:Alt Classe para conteiners de alternativas.
rdfs:Container Classe de containers RDF.
rdf:List Classe para listas de RDF.
https://www.w3.org/TR/rdf-schema/
